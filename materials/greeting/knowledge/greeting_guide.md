# MIT Framework - Product Overview

## What is MIT?

MIT (Multi Intelligence Twin) is an AI-powered multi-agent system that helps users find answers to technical questions across different domains. Each module is staffed by specialized AI agents with deep knowledge in their area.

## Available Modules

### Network Module
The Network module helps with networking-related questions, including:
- API endpoint documentation and usage
- Network error troubleshooting (504 timeouts, 401 unauthorized, etc.)
- Connection configuration and best practices

### Auth Module
The Auth module covers authentication and authorization topics:
- OAuth 2.0 flows and implementation
- Token management (JWT, opaque tokens)
- Authentication error diagnosis and resolution

## How to Use MIT

Simply ask your question in natural language. The system will:
1. Analyze your query to determine the best module
2. Route your question to the appropriate specialist
3. Retrieve relevant knowledge from the module's database
4. Synthesize a comprehensive answer

## Tips for Best Results

- Be specific about your problem or question
- Include error messages or codes when troubleshooting
- Mention the module or technology area if you know it
- Feel free to ask follow-up questions in the same conversation

## Support

If you're unsure which module handles your question, just ask! The system can suggest the right module for your needs.
