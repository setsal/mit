# Welcome to MIT! 🤖

**Multi Intelligence Twin** — A multi-agent framework for expertise-based Q&A.

Each query is routed to the best-fit **Module Agent**, which coordinates specialized **Sub-Agents** powered by RAG to retrieve and synthesize answers from your knowledge base.

## Available Modules

- **🌐 Network** — API endpoints, parameters, error codes, and troubleshooting
- **🔐 Auth** — Authentication flows, OAuth, and security configurations

## How It Works

1. 🔀 **Router** analyzes your query and selects the appropriate module
2. 🏷️ **Coordinator** classifies and dispatches to a specialized sub-agent
3. 🔍 **Sub-Agent** retrieves relevant documents from the knowledge base
4. 💡 **Synthesizer** produces a coherent final answer
